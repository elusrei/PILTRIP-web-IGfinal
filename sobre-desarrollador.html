Sobre el Proyecto

Este proyecto fue desarrollado como parte de la materia de Desarrollo Web. El objetivo principal fue crear un sitio web completo con múltiples secciones, subsecciones, formularios validados y elementos interactivos utilizando JavaScript.

Tecnologías Utilizadas

  • HTML5 para la estructura
  • CSS3 para los estilos
  • JavaScript para la interactividad
  • Bootstrap para el diseño responsive
  • Font Awesome para los iconos

El sitio web es completamente responsive y se adapta a diferentes tamaños de pantalla, desde dispositivos móviles hasta computadoras de escritorio.
